## What is Zeity?

Zeity is a modern, open-source time tracking application designed for individuals and teams. Built with [Nuxt 4](https://nuxt.com), [Vue 3](https://vuejs.org), and [PostgreSQL](https://postgresql.org), it provides a robust platform for tracking time, managing projects, and collaborating with teams.

## Key Features

### Time Tracking
- **Manual Time Entries** - Add time entries with start/end times or duration
- **Built-in Timer** - Start/stop timer for real-time tracking
- **Flexible Input** - Support for different time formats and timezones

### Project Management
- **Unlimited Projects** - Create and manage unlimited projects
- **Project Status** - Active and closed states
- **Color Coding** - Visual organization with custom colors

### Team Collaboration
- **Organizations** - Group users into organizations
- **Teams** - Create teams within organizations
- **Role-Based Access** - Owner, admin, and member roles
- **Invitations** - Invite users via email

### Reports & Export
- **Time Reports** - Filter by project, user, date range
- **Export Options** - CSV and Excel formats
- **Data Analysis** - Track productivity and billable hours

### Security & Authentication
- **Passkeys** - Modern passwordless authentication
- **OAuth Providers** - Google, Apple, Microsoft
- **Session Management** - Secure session handling
- **CSRF Protection** - Built-in security features

## Technology Stack

Zeity is built with modern, production-ready technologies:

- **Framework** - [Nuxt 4](https://nuxt.com) with Vue 3
- **Database** - [PostgreSQL](https://postgresql.org) 14+
- **ORM** - [Drizzle ORM](https://orm.drizzle.team)
- **Authentication** - [nuxt-auth-utils](https://github.com/atinux/nuxt-auth-utils)
- **UI** - [Nuxt UI](https://ui.nuxt.com) with Tailwind CSS
- **State Management** - [Pinia](https://pinia.vuejs.org)
- **PWA** - [@vite-pwa/nuxt](https://vite-pwa-org.netlify.app)

## Use Cases

### For Individuals
- Track personal projects and tasks
- Monitor time spent on different activities
- Generate productivity reports

### For Freelancers
- Track billable hours per client
- Generate invoices with time reports
- Manage multiple client projects

### For Teams
- Collaborate on shared projects
- Track team productivity
- Analyze resource allocation

### For Organizations
- Centralized time tracking
- Multi-team management
- Comprehensive reporting

## Architecture

Zeity follows a **monorepo structure** with shared libraries:

```
zeity/
├── apps/
│   ├── zeity/          # Main application
│   └── docs/           # This documentation
└── libs/
    ├── database/       # Database schemas
    ├── types/          # Shared TypeScript types
    └── utils/          # Utility functions
```
