## Track time with the timer

The built-in timer is the fastest way to track time in real-time.

### Start the timer

1. Click the **Timer** button in the navigation
2. Select a project from the dropdown
3. Add a description (optional)
4. Click **Start**

::note
The timer continues running even if you close the browser tab!
::

### Stop the timer

1. Click **Stop** when you're done
2. The time entry is automatically created
3. Edit the entry if needed

### Next task button

1. Click **Next Task** to stop the current timer and start a new one
2. Select a different project if needed
3. Add a description (optional)

## Add manual time entries

Create time entries after the fact.

### Add an entry with start/end times

1. Navigate to **Timer**
2. Click **Add Entry**
3. Select project
4. Enter start time (e.g., "9:00 AM")
5. Enter end time (e.g., "11:30 AM")
6. Add description (optional)

## Edit existing entries

Fix mistakes or add details to time entries.

### Edit an entry

1. Navigate to **Timer**
2. Click on the entry you want to edit
3. Modify any field

## Delete time entries

Remove incorrect or duplicate entries.

### Delete a single entry

1. Click on the time entry
2. Click **Delete** button

::warning
Deleted time entries cannot be recovered!
::

## Advanced Features

### Running timer indicator

A visual indicator shows when a timer is running:

- Badge on Timer button
- Time updates in real-time

### Multiple timers

::note
Only one timer can run at a time. Starting a new timer automatically stops the previous one.
::

### Timer persistence

Timers persist across:

- Browser refreshes
- Tab closures
