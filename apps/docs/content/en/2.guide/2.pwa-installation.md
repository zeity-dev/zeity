## Installation on Desktop

### Chrome / Edge / Brave (Windows, macOS, Linux)

1. **Open Zeity** in your browser
2. Look for the **install icon** (‚äï) in the address bar
3. Click the install icon
4. Click **Install** in the popup dialog

Alternatively:

1. Click the **three-dot menu** (‚ãÆ) in the top-right corner
2. Select **"Install Zeity..."** or **"Apps" ‚Üí "Install Zeity"**
3. Confirm the installation

The app will appear in:
- **Windows**: Start Menu and Desktop
- **macOS**: Applications folder and Dock
- **Linux**: Applications menu

### Firefox (Windows, macOS, Linux)

::note
Firefox on desktop does not support PWA installation. Use Chrome, Edge, or Brave for desktop PWA support.
::

### Safari (macOS)

::note
Safari on macOS does not fully support PWA installation. Use Chrome or Edge for the best desktop PWA experience.
::

## Installation on Mobile

### Chrome (Android)

1. **Open Zeity** in Chrome on your Android device
2. Tap the **three-dot menu** (‚ãÆ) in the top-right corner
3. Select **"Add to Home screen"** or **"Install app"**
4. Enter a name for the app (or keep "Zeity")
5. Tap **Add** or **Install**

The Zeity icon will appear on your home screen.

### Samsung Internet (Android)

1. **Open Zeity** in Samsung Internet
2. Tap the **menu icon** (three lines) at the bottom
3. Select **"Add page to"**
4. Choose **"Home screen"**
5. Tap **Add**

### Safari (iOS/iPadOS)

1. **Open Zeity** in Safari on your iPhone or iPad
2. Tap the **Share button** (square with arrow pointing up)
3. Scroll down and tap **"Add to Home Screen"**
4. Edit the name if desired
5. Tap **Add** in the top-right corner

The Zeity icon will appear on your home screen.

::tip
On iOS, only Safari supports PWA installation. Chrome and other browsers on iOS cannot install PWAs.
::

## Launching the Installed App

Once installed, you can launch Zeity like any other app:

- **Desktop**: Click the Zeity icon in your applications menu, Start menu, or Dock
- **Mobile**: Tap the Zeity icon on your home screen

The app will open in standalone mode without browser UI elements.

## Uninstalling the PWA

### Desktop

**Chrome / Edge / Brave:**
1. Open the installed Zeity app
2. Click the **three-dot menu** (‚ãÆ) in the top-right
3. Select **"Uninstall Zeity..."**
4. Confirm the uninstallation

Alternatively, right-click the app icon and select **Uninstall**.

### Mobile

**Android:**
- Long-press the Zeity icon on your home screen
- Drag to **"Uninstall"** or tap **"App info" ‚Üí "Uninstall"**

**iOS/iPadOS:**
- Long-press the Zeity icon on your home screen
- Tap **"Remove App"**
- Select **"Delete App"** and confirm

## Updating the PWA

PWAs update automatically in the background. When you launch the app:

1. The latest version downloads automatically
2. You'll see a notification when an update is available
3. Refresh or restart the app to use the new version

::tip
No manual updates required! Zeity updates seamlessly like a web app while giving you the benefits of a native app.
::

## Troubleshooting

### Install Button Not Appearing

**Check requirements:**
- Using a supported browser (Chrome, Edge, Brave on desktop)
- HTTPS connection is required
- Service worker is registered correctly
- Site is not already installed

**Try:**
- Clear browser cache and reload
- Check browser console for errors
- Use browser menu instead of install button

### PWA Not Working Offline

**Verify:**
- You've opened the app at least once while online
- Service worker is registered and active
- Visit `chrome://serviceworker-internals/` (Chrome) to check status

**Fix:**
- Reload the app while online
- Clear cache and re-register service worker
- Reinstall the PWA

### App Icon Not Updating

After reinstalling or updating:
- Clear browser cache
- Uninstall and reinstall the PWA
- Check manifest.json is loading correctly
